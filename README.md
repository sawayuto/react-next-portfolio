# react-next-portfolio

サイトURL：  
http://localhost:3000

## 課題の紹介

### TOP

- Next.jsを使用してポートフォリオサイトを作成しました。

- microCMSからデータを取得してサイトに表示しています。

### 自己紹介のページ

- microCMSで作成したプロフィールAPIから自己紹介情報を取得しています。

- 名前や自己紹介などの情報を表示しています。

### ブログ一覧ページ

- microCMSのブログAPIから記事一覧を取得して表示しています。

- 記事のタイトルや公開日を確認できます。

### ブログ詳細ページ

- ブログ記事をクリックすると詳細ページが表示されます。

- 記事の本文を確認することができます。

## 一番見てほしいところ

- microCMSとNext.jsを連携してデータを取得している部分です。

- APIから取得したデータをサイトに表示する仕組みを実装しました。
